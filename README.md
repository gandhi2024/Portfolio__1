# Prahlad Gandhi - Personal Portfolio

A modern, responsive personal portfolio website showcasing projects, skills, and experience as a full-stack web developer.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Hero Section** - Eye-catching introduction with call-to-action buttons
- **About Section** - Background and educational qualifications
- **Skills Showcase** - Organized technical skills and programming languages
- **Projects Gallery** - Displaying key projects with descriptions and technologies
- **Contact Form** - Functional contact section with form validation
- **Smooth Navigation** - Smooth scrolling and fixed header navigation
- **Mobile Menu** - Hamburger menu for mobile devices

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** CSS3 with CSS Variables, Responsive Grid & Flexbox
- **Icons:** Font Awesome 6.4.0
- **Fonts:** Google Fonts (Poppins, Montserrat)
- **Features:** Local Storage, DOM Manipulation, Smooth Scrolling

## Sections

1. **Header** - Navigation bar with responsive mobile menu
2. **Hero** - Main introduction with profile image and CTA buttons
3. **About** - Personal background, education, and career objectives
4. **Skills** - Technical skills organized by categories
5. **Projects** - Portfolio of completed projects with links
6. **Contact** - Contact information and contact form
7. **Footer** - Social media links and copyright

1. Open `index.html` in your web browser
